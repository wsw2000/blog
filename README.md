# blog

#### 介绍
  [博客线上地址](https://wsw2000.top/)  

  该博客使用的是react的ssr框架next.js，用到了hooks跟redux状态管理器，antd组件库，和天气api，canvas背景
  每日一句api，ip查询api，aplayer音乐插件，支持深色模式，明亮切换，支持响应式,博客日志分页及按照最新，热门排序未来还会继续完善
